## Building from Source

### Prerequisites

- [Nix: the package manager](https://nixos.org/download.html#nix-install-linux)

### Installation

Running `ltlr` is as simple as follows:

```bash
nix run --impure github:guibou/nixGL -- nix run github:delveam/ltlr
```
